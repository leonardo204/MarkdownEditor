Markdown Editor 설치 가이드
=============================

1. 설치
   MarkdownEditor.app을 Applications 폴더로 드래그하세요.

2. 기본 앱으로 설정 (선택사항)
   .md 파일을 Markdown Editor로 기본 열기:

   방법 1 - Finder에서 설정:
   • 아무 .md 파일을 우클릭
   • '정보 가져오기' 선택
   • '다음으로 열기'에서 MarkdownEditor 선택
   • '모두 변경...' 버튼 클릭

   방법 2 - 터미널에서 설정:
   $ brew install duti
   $ duti -s com.zerolive.MarkdownEditor .md all
   $ duti -s com.zerolive.MarkdownEditor .markdown all
